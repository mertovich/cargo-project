# Cargo Project

## API

|URL   |BODY   | METHOD   |
| ------------ | ------------ | ------------ |
| /api/registercargo  | owner, adress, location, status  | POST  |
| /api/getcargo  | id  | GET  |
| /api/updatecargolocation  |  id, location |  POST |
| /api/updatecargostatus  |  id, status |  POST |
